系统产品

    • 功能强大的实时总线分析工具

    • 支持A664，A661，A429，A629，M1553，A825，ASCB-D，P2P/IMB/ADB等总线

  2. 便携式飞行测试系统

    • 28V直流或110-240伏交流供电加固笔记本电脑，最多7个PCI扩展槽

    • 最多可叠加6套飞行测试系统，通过1台加固笔记本监控调度

    • 支持A664，A661，A429，A629，M1553，A825，ASCB-D，P2P/IMB/ADB等总线

  3. 实验室用飞行测试系统

    • 4U 6PCI 槽

    • 4U 12PCI槽

    • 配备加固型外框

  4. GSERV/GSIM

    • 可安装于桌面电脑或加固型电脑中的控仿真系统

    • GSERV：基于Windows的IO平台

    • GSIM：基于Linux的实时IO平台

智能板卡产品线
  1. ARINC 664/AFDX

    • 空客, 波音, 商飞, 湾流, 柯林斯, 霍尼韦尔，中航工业

  2. 飞控IMB/ADB总线

    • 波音787， 商飞C919

  3. ASCB-D/Enhanced

    • 支持霍尼韦尔 APEX EPIC航电系统

  4. ARINC 429

    • 32通道PCI，PMC， PCIe， 16通道IP

    • 任意通道均可由软件配置为Tx或Rx状态

  5. PCI 接口

    • 8通道热电偶， 波形发生卡

  6. IP 卡

    • 32通道，28伏 GDO离散输入/输出

    • 24通道， 大电流离散输入/输出

    • 16通道RS485，Tach fuel流量计

    • 25通道笛簧继电器

    • 4通道模拟倍频器

    • 4通道IOAC GP500总线

    注：Goebel全球独创的智能板卡，板载处理资源可代替上位机程序执行故障注入，EDE校验等高实时性需求任务。

标准软件产品
  1. Goebelyzer/Gtools ANALYZE

    • 功能强大的实时总线分析工具

    • ICD导入器，可根据ICD文件直接对总线数据进行解码

  2. RECORD

    • 在保证数据同步的前提下，记录各种总线上的传输数据

  3. PASSTHRU

    • 故障注入工具，通过Goebel智能板卡修改总线数据，进行故障仿真。

    • 测试期间，客户应用软件中无需添加故障注入代码

  4. REPLAY

    • 回放Goebelyzer捕获的数据或NTAR文件数据

  5. SCRIPTING

    • 测试用例编辑器

  6. DATALOADER

    • 成功运用于787和空客相关项目中

  7. GIO IO server

    • ICD 驱动IO

    • 支持测试脚本

    • REST/JSON 协议

测试台模块式核心机

Goebel提供专业的软硬件标准模块，可应用于航电及飞控系统开发的各阶段。

Goebel测试台核心模块充分利用了Goebel智能板卡的特点，优化了系统实时性，降低了上位机软件复杂度，缩短了设备开发周期。

系统集成商可在Goebel标准模块基础上，按照最终用户的项目要求进行二次开发。

此模式可大大提高开发速度并减少设备故障率。
